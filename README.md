# 🔐 SezarConsole – Console-based Caesar Cipher Tool

Welcome to **SezarConsole**, a simple console-based application that encrypts your text using the classic Caesar cipher method. This project is great for learning basic C# skills like string manipulation, user input, and logic flow.

---

## 🧠 What does it do?

When you run the app, you'll see a welcome message and be prompted to:

- Enter a word or sentence to encrypt  
- Choose how many characters to shift (e.g., 3)

The app will then encrypt your input using Caesar cipher logic and display the result on the console.

---

## 🛠 Technologies Used

- C# (.NET)  
- Console Application

---

## 🚀 How to Run

To run this project locally on your machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/SezarConsole.git
   ```

2. Open the project in Visual Studio or VS Code

3. Make sure you have the .NET SDK installed

4. Build and run the project:
   ```bash
   dotnet run
   ```

---

## 📌 Features

- ✏️ Encrypts any text using Caesar cipher  
- 🔢 User-defined shift amount  
- 🔤 Supports both uppercase and lowercase letters  
- 🔒 Leaves non-alphabetic characters unchanged

---

## 🧪 Example

```
Input: hello world
Shift: 3  
Output: khoor zruog

```
